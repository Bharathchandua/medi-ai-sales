# MediAI Sales

MediAI Sales is a modern Java backend project for medical sales, inventory, billing, and AI-assisted business workflows.

## Current Goal

This project is being built as a job-ready Java backend portfolio project:

- Java 17
- Spring Boot 3
- REST APIs
- Spring Data JPA
- MySQL
- Validation
- Clean controller-service-repository structure
- Swagger/OpenAPI
- Tests
- AI integration in later milestones

## Planned AI Features

- Ask questions about stock in natural language
- Summarize invoices
- Suggest low-stock and expiring products
- Upload product/vendor documents and ask questions using RAG

## Day 1 Learning Focus

Understand and explain this flow:

```text
Controller -> Service -> Repository -> Database
```

For every API, know:

- What request comes from the client
- Which DTO receives the request
- Which service method contains business logic
- Which repository talks to the database
- What response goes back
